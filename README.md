# Blockbound

**A Terraria-inspired 2D mining, crafting & exploration game** made for GameDev.js Jam 2026.

Dig deep, craft tools, build shelters, and survive the dangers lurking beneath the surface.

### 🎮 Play the Game
→ [Play Blockbound on itch.io](https://your-username.itch.io/blockbound)  
*(Link will be added after submission)*

### ✨ Features
- Destructible block-based world
- Mining and resource gathering
- Tool crafting system
- Building and base creation
- Day/night cycle (planned)
- Enemies and underground dangers
- Smooth pixel-art style with physics

### 🎯 Controls
- **Arrow Keys / WASD** — Move
- **Space** — Jump
- **Mouse Left Click** — Mine / Break blocks
- **Mouse Right Click** — Place blocks
- **Number Keys (1-9)** — Select hotbar items

### 🛠️ Built With
- Phaser 3 (JavaScript)
- HTML5 Canvas
- Procedural world generation
